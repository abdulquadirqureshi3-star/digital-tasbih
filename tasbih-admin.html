🚀 Firebase + Play Store Setup
Step by step — mostly free
1
Firebase Project Banao
console.firebase.google.com → "Add project" → naam: digital-tasbih → Google Analytics ON karo → Create Project
2
Realtime Database Enable Karo
Firebase Console → Build → Realtime Database → Create Database → Location: asia-southeast1 (Singapore) → Start in test mode → Enable
URL milega: https://digital-tasbih-default-rtdb.asia-southeast1.firebasedatabase.app
3
Web App Add Karo — Config Copy Karo
Project Overview → "Add app" → Web icon → nickname: tasbih-web → Register → firebaseConfig copy karo → Project Settings mein jaake Web API Key bhi note karo
firebaseConfig = { apiKey: "AIzaSy...", ← Web API Key databaseURL: "https://...", ← sabse zaroori projectId: "digital-tasbih", ... }
4
digital-tasbih.html Mein Config Daalo
HTML file mein ye section dhundo:
const FIREBASE_CONFIG = { apiKey: "APNI_VALUE", authDomain: "APNI_VALUE", databaseURL: "APNI_VALUE", projectId: "APNI_VALUE", storageBucket: "APNI_VALUE", messagingSenderId: "APNI_VALUE", appId: "APNI_VALUE" };
5
Admin Panel Mein Firebase Config Daalo
Is admin panel → Settings tab → Firebase Database URL aur Web API Key paste karo → Save → "Test Firebase" se verify karo
6
Database Security Rules Set Karo
Firebase Console → Realtime Database → Rules:
{ "rules": { ".read": true, ".write": false } }
Public read (app ke liye), write sirf Console se hoga.
7
GitHub Pages Pe Host Karo (Free)
github.com → New repo: digital-tasbih → Public → index.html naam se upload → Settings → Pages → main branch → Save
URL: USERNAME.github.io/digital-tasbih
8
manifest.json Upload Karo
{ "name": "Digital Tasbih", "short_name": "Tasbih", "start_url": "/", "display": "standalone", "theme_color": "#c9a84c", "background_color": "#0a0c10", "icons": [ {"src":"icon-192.png","sizes":"192x192","type":"image/png"}, {"src":"icon-512.png","sizes":"512x512","type":"image/png"} ] }
9
Android App Banao — PWABuilder (Free)
pwabuilder.com → GitHub URL daalo → Package for Android → Download AAB
10
Play Store Publish ($25 ek baar)
play.google.com/console → $25 account → Create App → AAB upload → Submit → 3-7 din

Admin Panel URL: tasbih-admin.html bhi GitHub mein upload karo.
URL: USERNAME.github.io/digital-tasbih/admin.html

Firebase Free Limits: 1GB Storage + 10GB/month bandwidth = lakhon users ke liye kaafi!
Total Cost: Firebase Free + GitHub Free + PWABuilder Free + Play Store $25 = Sirf $25!
